# Payment-Trash-API

## Clone & Run
Clone this project
```
https://github.com/ASNProject/nodejs-payment-trash-api.git
```
Run program
```
nodemon server.js
```
Download Database
```
/mysql_database/db_payment_trash_api.sql
```
